# csharp-lsp

用于 Claude Code 的 C# 语言服务器，提供代码智能和诊断功能。

## 支持的扩展名

`.cs`

## 安装

### 通过 .NET 工具（推荐）
```bash
dotnet tool install --global csharp-ls
```

### 通过 Homebrew（macOS）
```bash
brew install csharp-ls
```

## 要求

- .NET SDK 6.0 或更高版本
