# ruby-lsp

用于 Claude Code 的 Ruby 语言服务器，提供代码智能和分析功能。

## 支持的扩展名

`.rb`, `.rake`, `.gemspec`, `.ru`, `.erb`

## 安装

### 通过 gem（推荐）
```bash
gem install ruby-lsp
```

### 通过 Bundler
添加到您的 Gemfile：
```ruby
gem 'ruby-lsp', group: :development
```

然后运行：
```bash
bundle install
```

## 要求

- Ruby 3.0 或更高版本
