# lua-lsp

用于 Claude Code 的 Lua 语言服务器，提供代码智能和诊断功能。

## 支持的扩展名

`.lua`

## 安装

### 通过 Homebrew（macOS）
```bash
brew install lua-language-server
```

### 通过包管理器（Linux）
```bash
# Ubuntu/Debian（通过 snap）
sudo snap install lua-language-server --classic

# Arch Linux
sudo pacman -S lua-language-server

# Fedora
sudo dnf install lua-language-server
```

### 手动安装
从 [releases 页面](https://github.com/LuaLS/lua-language-server/releases) 下载预编译的二进制文件。
