# swift-lsp

用于 Claude Code 的 Swift 语言服务器（SourceKit-LSP），为 Swift 项目提供代码智能功能。

## 支持的扩展名

`.swift`

## 安装

SourceKit-LSP 随 Swift 工具链一起提供。

### macOS
从 App Store 安装 Xcode，或通过以下方式安装 Swift：
```bash
brew install swift
```

### Linux
从 [swift.org](https://www.swift.org/download/) 下载并安装 Swift。

安装后，`sourcekit-lsp` 应该在您的 PATH 中可用。
