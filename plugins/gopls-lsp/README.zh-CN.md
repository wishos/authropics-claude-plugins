# gopls-lsp

用于 Claude Code 的 Go 语言服务器，提供代码智能、重构和分析功能。

## 支持的扩展名

`.go`

## 安装

使用 Go 工具链安装 gopls：

```bash
go install golang.org/x/tools/gopls@latest
```

确保 `$GOPATH/bin`（或 `$HOME/go/bin`）在您的 PATH 中。
