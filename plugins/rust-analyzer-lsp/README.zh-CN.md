# rust-analyzer-lsp

用于 Claude Code 的 Rust 语言服务器，提供代码智能和分析功能。

## 支持的扩展名

`.rs`

## 安装

### 通过 rustup（推荐）
```bash
rustup component add rust-analyzer
```

### 通过 Homebrew（macOS）
```bash
brew install rust-analyzer
```

### 通过包管理器（Linux）
```bash
# Ubuntu/Debian
sudo apt install rust-analyzer

# Arch Linux
sudo pacman -S rust-analyzer
```

### 手动下载
从 [releases 页面](https://github.com/rust-lang/rust-analyzer/releases) 下载预编译的二进制文件。
